# CTUnlockable

CTUnlockables exist for both custom unlockables, and traits built into Reskillable.

| ZenGetter/ZenMethod | ZenSetter/ZenMethod | Type     |
|---------------------|---------------------|----------|
| name                |                     | string   |
| getDescription      |                     | string   |
| parent              |                     | CTSkill  |
| enabled             | enabled             | boolean  |
|                     | requirements        | string[] |
| icon                |                     | string   |
| cost                | cost                | int      |
| x                   | x                   | int      |
| y                   | y                   | int      |
| spikes              |                     | boolean  |
